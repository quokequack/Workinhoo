# Workinhoo 🧰
Marketplace de trabalhos informais

Os trabalhadores informais - pintores, costureiras, encanadores e etc. - encontram dificuldades na divulgação de seus serviços, dependendo unicamente de uma rede de indicações 'boca a boca'. Com isso, o Workinhoo surge como uma iniciativa para facilitar a promoção desses serviços, tanto para quem faz quanto para quem precisa, e não sabe quem contatar.

Mais informações na wiki do projeto: https://github.com/quokequack/Workinhoo/wiki
